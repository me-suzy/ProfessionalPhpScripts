Back Up your Database

 Nothing is worse then having your precious database of link partners nuked.

This is in place to protect you incase the whole database goes down. Then in the event that something goes wrong you can use the backed up copies.

What this program does is copy the entire database to es/backup/
Whenever you backup again, it merely rewrites the old backup files.

Since it backs up the entire database, please reset the IP file first so it doesn't have to copy that when it is so large.
