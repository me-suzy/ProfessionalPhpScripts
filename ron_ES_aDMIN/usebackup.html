Use backup for a single partner:
-You can use this is a partners stats file is empty.

OR

 Use entire back up database

THIS IS NOT TO BE USED UNLESS IT IS A VERY SERIOUS PROBLEM!

- If you use this it will work off of your back up aU.dat. This is not recommended unless you have major corruption. If you do this then you may have stats files lingering around from old partners. So if you try to add them again, you may get an error and have to run "delete" for that user before you add them again. After you do this, run "the doctor" and then after you fix that, then run the "clear hits".
