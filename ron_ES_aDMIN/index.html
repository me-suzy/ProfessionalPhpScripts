ES v1.0 Admin Interface

Statistics: 

 View Stats:

  view realtime stats for all users.
       

  Partner Utilities:

Add Partner:

  add new partner with inhits, outhits, referring URL

 Edit Partner:

  edit current partner's inhits, outhits, and referring URL

 Delete Partner:

  completely removes a partner from ES.
       

 Return Traffic:

 Default Option:

  set ES to return only a 1:1 ratio, to a percentage return, or to a surge.

 Random Return:

  define the ratios to return to partners if the default option is set to "random return percentage".
       

Manual Utilities:

 Clear Hits:

 resets all the users stats to either what they are owed or to 10 as to stoke the traffic exchange.

 Reset Uniques:

 clears the unique file which new hits are compared against before they are tracked.

 Default URL:

  sets an URL which a user will be redirected to in case of an error, or if the "default option" is set to either 1:1 or percentage return and all the users have been paid off for the hits they are owed.

Black List:

  add users to the black list

 Backup Database

 copies the entire database and saves it to es/backup 

 Run the doctor

  determines if your ES program is healthy or not.

Use backup 

  if you are having problems, use this to set all the users/ or a single user back to what they were when you last ran the backup program. Be careful using this.
