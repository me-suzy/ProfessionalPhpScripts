PHP Banner Network

1) Upload all the files to you webspace.

2) CHMOD users.txt, conf.php and counters to 666 (owner Read/Write, group Read/Write, public Read/Write)
3) Use your /htaccess file to password protect adduser.php
4) Goto the url of your banner network and the install page should load.
6) Install Your banner network
7) Delete install.php to prevent overwriting you choices
