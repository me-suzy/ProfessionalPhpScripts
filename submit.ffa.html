SimpleSubmit! FFA Links Page Example

This page provides an example for adding Free For All Links pages to your submission database. If you open submit.tab using a text editor, you'll see that there is an engine called VerySimpleFFA. (see submit.readme.html for more information about editing the .tab file)

In addition to editing the tabfile, you'll need to create a custom front page to the script that has all of the extra variables needed by your FFA links page. In the example below, Title and Section are needed by the FFA links page in addition to the usual URL and Email fields.
