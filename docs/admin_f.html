FUNCTION DESCRIPTIONS





    

  1.  Re-rank The List

    
This function will re-rank the list using the current data in the database.


    


  2.  Review New Signups

    
This function will allow you to review new signups if you choose that option
in the setup.cgi file.  Accepting a site will add them to the active database
and send an e-mail to them telling that their site has been activated. Declining
a site will simply delete it from the inactive database.


    


  3.  Display All Members

    
This function will display all active members currently in your database.


    


  4.  Edit List Breaks

    
This function will allow you to edit the HTML that will appear in the places
you have indicated that you want the list to break.


    


  5.  Edit Your Setup

    
This function will bring up the setup.cgi form and allow you to change all the
variables for your list.


    


  6.  Change Admin Password

    
This function will allow you to change the adminsitrative password.


    


  7.  Send E-Mail To Members

    
This function will allow you to send e-mail to all members.


    


  8.  Ban A Site

    
This function will allow you to ban a URL from signing up.
To ban an entire domain enter:
domain.com
To ban an exact URL enter the exact URL:
http://www.site.com/~user/index.html


    


  9.  Display Error Log

    
This function will display the entire contents of the error log if you
chose to log all errors in the setup.cgi script.


    


  10.  Keyword Search

    
This is a very simple search engine.  It will search all fields of every member's
data for the keyword entered and make a list of matches.
