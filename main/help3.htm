Framefree Mode


If you're having trouble reading the messages, convert to Framefree mode.
Click your mouse on the top frame and drag it almost all the way to the bottom.

The framefree chat does not autorefresh,
so you'll need to hit [Update] periodically to see new messages.  
However, this mode is less likely to cause errors.

To return to frames mode, drag the frame back up, and click link below.

[Return To Chat Window]

NOTE: Netscape 2 users!  To use this mode you must:
  1. Click Options on top menu bar.

  2. Select General Preferences.

  3. Choose the Colors Tab.

  4. Check "Alway Use My Colors, Overriding Document."
